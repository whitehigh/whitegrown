# douyin-python

Tools for exporting Douyin Creator Center work metrics, filtering works by play count, downloading selected videos, and capturing native backend screenshots.

## What is included

This folder is prepared for the Douyin Creator Center automation scripts from the local project at `D:\爬数据`.

The local project contains scripts for:

- opening Douyin Creator Center with a persistent browser profile
- exporting work metrics for a date range
- filtering works by play count
- downloading selected videos
- capturing native backend work-card screenshots

## Privacy

The following local paths are intentionally not uploaded:

- `.douyin-profile/` browser login state
- `node_modules/`
- `out/` exported data, screenshots, and downloaded videos
